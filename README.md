# Sky Islands
Hack and slash game with a [procedurally generated map][generator] - Created using C++ 14 and SFML 2.5.1

## Controls
- WSAD - movement
- ↑→↓← - shoot
- Space - generate new map
- ESC - exit
- For testing: Q & E - zooming, tab - show FPS
